## 玩法
​
> 在万圣节的午夜钟声敲响时，现实世界与数据帷幕之间的界限变得最为薄弱。一个古老的数据裂隙被撕开，成群的“代码幽灵”涌入了互联网。这些幽灵以网页的结构和内容为食。它们会“附身”  随机的HTML元素（如按钮、图片、段落），被附身的元素会开始抖动 、旋转 、闪烁  或扭曲。你是一名“数据驱魔师”  或 “网络净化者” 。你（的光标）是唯一能看到并对抗这些幽灵的存在。在幽灵们彻底吞噬整个页面、导致其“数据崩溃” 之前，净化它们，关闭裂隙。​
​
安装[用户脚本](https://raw.githubusercontent.com/chrome-haunted/chrome-haunted/refs/heads/main/index.js) 或者从下面下载 Chrome 扩展。点击任意网页的右下角的👻图标，开始游戏！​
